# UI Kit — ONE Collect

Recreation of the PGMais phygital collection dashboard. Navy dashboard chrome, bright metric cards, the green accent for active filters + primary actions.

## Components
- `Sidebar.jsx` — left rail with PG+ mark + nav
- `TopBar.jsx` — search, filters, user menu
- `KpiRow.jsx` — 4 metric tiles
- `JourneyTable.jsx` — active collection journeys list
- `AiPanel.jsx` — IA insights rail on the right
- `ConversationPreview.jsx` — channel chat thread preview
