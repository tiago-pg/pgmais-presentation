# UI Kit — PGMais Marketing Site

Recreation of `pgmais.com.br` style marketing surface.

Design language: **navy backgrounds (#172c66)** with gradient blob heroes (green→vibrant-blue), light sections (#ededed) for text density, and the PGMais verde (#40eb4f) reserved for primary CTAs and high-impact accents only. Lato for headlines (Black/Bold), Poppins for body + CTAs.

## Components
- `Nav.jsx` — glass-morphic sticky nav, Wordmark on light, Lettermark on dark
- `Hero.jsx` — navy backdrop with gradient blob + headline + dual CTA
- `EcosystemGrid.jsx` — 6 cards describing the ecosystem pillars
- `StatsBand.jsx` — 4-up KPI row
- `OneCollectFeature.jsx` — product feature block (single phygital system)
- `CaseLogos.jsx` — client logo wall
- `Footer.jsx` — navy footer with links + contact
