Components · Buttons

Buttons

Componente de botões preenchidos e com ícones. Os botões representam ações que o usuário pode tomar — desde a ação principal de uma tela até interações secundárias e neutras. A hierarquia visual reflete a importância da ação.

Anatomia

Estrutura base: container + label + ícone (opcional). Padding e raio são consistentes entre as variantes preenchidas.
